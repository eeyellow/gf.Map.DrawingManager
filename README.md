gf.TreeLayer.js
===========

**A jQuery plugin to generate hierarchy layer list** 


How to Use
--------
```html
<div id='myDiv'></div>
```

```js
$('#myDiv').gfTreeLayer(settings);
```

Settings
--------
```js
{
    arrData: slayer.arrData,    
    nameField: 'name',
    filterable: true,
    searchable: true,
    collapsible: false,
    hideable: false,
    openAll: true,   
    numFormat: { mode: [0] },             
    css: {
        'titleBackgroundColor': '#FF6600',
        'width': '250px',
        'height': '250px'
    }
}
```
