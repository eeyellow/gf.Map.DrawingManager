gf.Map.DrawingManager.js
===========

**A jQuery plugin which wrap Google Map drawing manager ** 

Dependencies
--------
1. jQuery
2. Google Map Javascript API

How to Use
--------
```html
<div id='myDiv'></div>
```

```js
var map = new google.maps.Map(...);

$('#myDiv').gfMapDrawingManager({
    targetMap: map    
});
```
